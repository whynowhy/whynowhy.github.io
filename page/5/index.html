访问localhost:8080时出现Whitelabel Error Page

需要输入在类中输入注解@RequestMapping("/")来映射地址，为了使用@RequestMapping("/")，必须在头部导入org.springframework.web.bind.annotation.RequestMapping包。

@RestController注解等价于@Controller+@ResponseBody的结合，使用这个注解的类里面的方法都以json格式输出，必须在头部导入import org.springframework.web.bind.annotation.RestController包。

若未导入包前已启动项目，需要用maven reimport更新依赖包，不然会报错。

博客的编写与发布

发表于 2018-11-21 更新于 2019-07-28 分类于 Blog

博客的创建

在命令行中进入博客的文件夹， 使用hexo new "博客的名字"创建一篇博客。

博客的编写

创建之后博客的md文件存储在/blog/source/_post中，编辑相关md文件即可编写博客。注：tag的形式为[tag1,tag2,tag3……]。

未发布博客的查看

使用hexo server即可启动本地服务器，通过输入http://localhost:4000即可进入本地的博客浏览编写的博客。

发布博客

使用hexo deploy -git即可发布博客。

关于这个博客的搭建

发表于 2018-11-09 更新于 2019-07-28 分类于 Blog

一、博客的搭建

整个博客的搭建可以从这里入手，接下来我会讲一讲我在搭建博客时遇到的问题

1.hexo安装时的问题

遇到了ETIMEOUT的报错，这时只要将npm的安装源换到国内就行了，方法是在命令行中输入如下：
npm config set registry https://registry.npm.taobao.org
之后输入一下内容进行测试：
npm config get registry

2.生成ssh key的正确指令

我引用的博客中写的是:
ssh-keygen -t- rsa -C "your email"
其实应该是：
ssh-keygen -t rsa -C "your email"
