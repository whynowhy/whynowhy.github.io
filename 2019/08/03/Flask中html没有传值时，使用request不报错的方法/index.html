Flask中html没有传值时，使用request不报错的方法

发表于 2019-08-03 更新于 2020-09-16 分类于 Flask

之前使用request.args['']的方法来获取html的传值。但是使用这种方法传递数据时，如果没有传值网站会报404的错误。经过调查之后发现，使用request.args.get('')来获取传值时，如果没有传值，就不会报错，返回的数据为None。
