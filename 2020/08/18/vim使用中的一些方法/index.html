vim使用中的一些方法

发表于 2020-08-18 分类于 Linux

vim中的设置

vim的用户配置文件应该是在/usr/bin/vim/.vimrc中，但是本人配置了.vimrc后不起作用，所以对系统的vimrc文件/etc/vimrc进行修改。

syntax enable “自动检测语法
syntax on “自动语法高亮
set number “显示行号
set tabstop=4 “表示Tab代表4个空格的宽度
set expandtab “表示Tab自动转换成空格
set autoindent “表示换行后自动缩进
set history=400 “vim记住的历史操作的数量，默认的是20
set smartindent “智能对齐
set shiftwidth=4 “偏移4个空格
set cursorline “每行都有下划线

不用箭头键进行上下左右移动光标

h是左移，l是右移，j是下移，k是上移

上下翻页

pageup 上翻页，pagedown 下翻页

进入输入模式

a 光标后插入，i 光标前插入， o 下一行插入

删除

单行删除 dd，全文删除 dG

复制

单行 在命令模式下，将光标移动到将要复制的行处，按“yy”进行复制，将光标移动到将要粘贴的行处，按“p”进行粘贴。

多行 在命令模式下，按v进入可视模式，按箭头键即可选择相应的行，选择相应的行之后，到复制的文本结束的地方，再按y进行复制，之后会自动退出退出可视模式，光标返回到文本开始的位置。移动光标到需要开始复制的地方，按p进行复制
