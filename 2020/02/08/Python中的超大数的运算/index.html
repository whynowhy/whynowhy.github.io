gmpy2包

经过查阅之后，发现使用gmpy2包可以解决这个问题，gmpy2是一个由C语言编码的Python扩展模块，支持多精度算术。 gmpy2是原始gmpy模块的后继者。 gmpy模块仅支持GMP多重精度库。 gmpy2增加了对MPFR（正确舍入的实数浮点运算）和MPC（正确舍入的复数浮点运算）库的支持。 gmpy2还更新了API和命名约定，使其更加一致并支持其他功能。

gmpy2包在pip安装时可能会因为系统和版本的原因报错，推荐直接上https://github.com/aleaxit/gmpy/releases上选择对应python版本和系统的包下载。
